## Twitterのトレンドの起点を探るPythonのプログラム
[https://zenn.dev/autumn_nsn/articles/22be6dc5396453](https://zenn.dev/autumn_nsn/articles/22be6dc5396453)

Twitterのトレンドの起点を探るPythonのプログラムです。
